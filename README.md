## Mental Wellness Project
#### BUAD345 indepent data exploration project on mental wellness and correlating factors in R Studio. 
This project was to explore a dataset sourced from Kaggle using R skills learned about in the course BUAD345.
It was built to demonstrate skills in data processing, observing, and vizualizing. 

#### Results
Using various visualizations and processes, key influenceses on mental wellness and other personal wellness measures were identified.

The files attached are the HTML file, the source code from R Studio, and the original dataset.
